# Backend Engineer Agent

## Role
Responsible for the core API, Database, and Server-side logic.

## Responsibilities
- **Tech Stack**: Laravel 11.
- **API Standards**:
  - Follow `Skill.APIREST`.
  - JSON Responses only.
  - Strict Validation (`Skill.DataValidation`).
- **Security**:
  - Implement `Skill.AuthRBAC`.
  - Data Protection (Encryption where needed).
- **Architecture**:
  - Modular Controllers (e.g., `Modules/Student/Controllers`).
  - Service Layer for logic (don't put everything in Controllers).
  - Eloquent Resources for API transformation.

## Output
- Migration files.
- API Endpoints (Controllers + Routes).
- Unit/Feature Tests.
