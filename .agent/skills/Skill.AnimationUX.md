# Skill.AnimationUX

## Objective
Delivering a "Premium" feel via motion.

## Instructions
1. **Tools**: Framer Motion (React) / CSS Transitions.
2. **Principles**:
   - **Subtle**: No jarring movements.
   - **Feedback**: Buttons verify clicks (ripple or scale).
   - **Transitions**: Page loads should fade/slide in.
3. **Standard Durations**:
   - Micro: 0.2s (hover).
   - Macro: 0.4s (page transition).
