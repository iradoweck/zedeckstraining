# Skill.DataValidation

## Objective
Ensure data integrity and security at all layers.

## Instructions
1. **Frontend**:
   - Real-time feedback (e.g., "Email is invalid").
   - Prevent form submission if invalid.
2. **Backend (CRITICAL)**:
   - Re-validate EVERYTHING.
   - Use Form Requests in Laravel.
   - Sanitize inputs.
3. **Database**:
   - Use Foreign Key constraints.
   - Use suitable data types.
