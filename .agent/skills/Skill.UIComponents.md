# Skill.UIComponents

## Objective
Ensure visual consistency and reusability across all apps.

## Instructions
1. **Library**: Create a shared component folder (e.g., `src/components/ui/`).
2. **Core Components**:
   - `Button` (Primary, Secondary, Outline, Danger).
   - `Card` (Container with standard padding/shadow).
   - `Input` / `Select` (Standardized form elements).
   - `Modal` (Accessible dialog).
   - `Table` (Clean data display).
3. **Styling**:
   - Use TailwindCSS classes.
   - Use CSS Variables for colors (`var(--color-primary)`).
4. **Interactive**:
   - All interactive elements must have hover/focus states.
