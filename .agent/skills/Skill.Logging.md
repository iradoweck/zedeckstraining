# Skill.Logging

## Objective
Traceability and Debugging.

## Instructions
1. **Activity Logs**:
   - Record WHO did WHAT and WHEN.
   - Context: "Admin X updated Grade for Student Y entirely".
2. **Error Logs**:
   - Stack traces for 500 errors.
   - Notification for critical failures.
3. **Storage**:
   - DB for Activity Logs (audit trail).
   - File/External for System Errors.
